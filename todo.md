Create a learning backend using Java 21 and Spring Boot 3 for a REST API that manages orders.

The application should include:
- JWT-based authentication
- RBAC roles: USER and ADMIN
- PostgreSQL database
- Flyway database migrations
- Centralized error handling
- Audit logging
- Actuator health endpoints
- Docker Compose setup for local development
- Integration tests

The goal of the project is not production-ready business logic, but to demonstrate a secure-by-default approach and cover protection practices against OWASP Top 10.
